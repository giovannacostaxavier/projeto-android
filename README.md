# 📱 Projeto Android – Curso de HTML e CSS

Este projeto é uma página web desenvolvida durante as aulas do curso de **HTML e CSS** do canal [Curso em Vídeo](https://www.youtube.com/c/CursoemVideo).  

O objetivo é aplicar conceitos de **estrutura, estilização e layout** em um site simples, mas funcional, integrando imagens, fontes e estilos personalizados.

---

## 🌐 Demonstração

🔗 Acesse o projeto online:  
👉 [https://giovannacostaxavier.github.io/projeto-android/](https://giovannacostaxavier.github.io/projeto-android/)

---

## ✨ Sobre o Projeto

O site apresenta:

- Layout básico com **HTML5**  
- Estilização com **CSS3**  
- Uso de **imagens e fontes personalizadas**  
- Estrutura organizada em seções, tornando a navegação clara  
- Design simples e harmônico, focado em prática e aprendizado  

Este projeto é um exemplo de como combinar **conteúdo textual e visual** de maneira coerente, aplicando conceitos fundamentais de front-end.

---

## 🛠 Tecnologias Utilizadas

- **HTML5** – Estrutura da página  
- **CSS3** – Estilização e layout  
- **Visual Studio Code** – Editor de código  
- **Git & GitHub** – Controle de versão e hospedagem  

---

## 📂 Estrutura do Projeto

- `index.html` → Página principal  
- `style.css` → Estilos e layout do site  
- `imagens/` → Imagens utilizadas  
- `fontes/` → Tipografias personalizadas  


---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:

- Praticar **HTML e CSS** na criação de páginas web  
- Trabalhar com **imagens, fontes e estilização**  
- Organizar conteúdo de forma clara e visualmente agradável  
- Consolidar conhecimentos de front-end aprendidos no curso
